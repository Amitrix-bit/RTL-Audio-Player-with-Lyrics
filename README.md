## ✨ ویژگی‌های کلیدی  
- **نمایش متن آهنگ** با هایلایت همزمان  
- **تایم‌لاین قابل کشیدن** برای جستجوی دقیق  
- **طراحی واکنش‌گرا** (مورد پسند موبایل و دسکتاپ)  
- **تغییر تم خودکار** (تاریک/روشن)  
- **پخش خودکار** با قابلیت کنترل از رسانه‌های سیستم
